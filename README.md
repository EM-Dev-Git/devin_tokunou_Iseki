# 井関物流 受払管理システム (Iseki Logistics Receipt and Payment Management System)

## 概要 (Overview)
このリポジトリは井関物流向けの受払管理システム（IbUkeharai）を含んでいます。本システムは部品の受入・払出の管理、在庫管理、請求書発行などの機能を提供します。

## システム構成 (System Structure)
システムは以下の3つのプロジェクトで構成されています：

### IbUkeharai（メインアプリケーション）
- Windows Formsアプリケーション
- 受払業務の主要機能を提供
- ユーザーインターフェースとビジネスロジックを含む

### DitCore（コアライブラリ）
- 共通クラス、ダイアログ、ユーティリティを提供
- メッセージボックス、進行状況ダイアログなどのUI要素
- 設定ファイル処理、ログ出力などの基本機能

### DitDataAccess（データアクセスライブラリ）
- データベースアクセス機能
- SQLServerとの連携
- ファイル入出力操作

## 主要機能 (Main Features)

### 登録機能 (Registration)
- 受入実績登録
- 払出実績登録
- 取引先マスター登録
- 部品マスター登録
- 単価マスター登録
- 請求マスター登録
- 納入先マスター登録

### 問合せ機能 (Inquiry)
- 受払実績照会
- 在庫照会

### バッチ処理 (Batch Processing)
- 受入データ書き込み
- 払出データ書き込み
- 月次データ処理
- 年次データ処理
- 取引先一括変更

### データ作成 (Data Creation)
- 取引先別受払データ作成（日別）
- 取引先別受払データ作成（月別）
- 月刊受払日別展開データ作成
- 部品マスターデータ作成
- 取引先マスターデータ作成
- 納入先マスターデータ作成

### リスト出力・請求 (Report Output & Billing)
- 受払報告書
- 受払確認書
- 在庫注残リスト
- 部品別受払プリント
- 実績内訳表出力
- 請求書出力

## 技術情報 (Technical Information)
- 開発言語: VB.NET
- UI: Windows Forms
- データベース: SQL Server
- 対象フレームワーク: .NET Framework 4.0
- 開発環境: Visual Studio

## セットアップ (Setup)
1. リポジトリをクローン
2. Visual Studioで`IbUkeharai.sln`を開く
3. 必要なNuGetパッケージを復元
4. アプリケーションをビルド
5. 設定ファイルでデータベース接続などを構成
6. アプリケーションを実行

## 注意事項 (Notes)
- 本システムは井関物流向けに開発されたもので、特定の業務フローに最適化されています
- データベース接続情報は適切に構成する必要があります
