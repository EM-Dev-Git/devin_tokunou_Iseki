# 請求処理フロー 改修版

## 概要
本ドキュメントは、「請求処理フロー.xlsx」の改修版です。請求マスター登録時の自動計算機能追加および入力チェック強化、請求書出力時の単位情報取得に関する変更を反映しています。

## 1. 請求マスター登録フロー

### 変更前
```
[開始] → [請求マスター登録画面表示] → [取引先コード入力] → [取引先マスター検索] → [部品コード入力] → [部品マスター検索] → [数量入力] → [単価入力] → [金額入力] → [登録ボタン押下] → [入力チェック] → [DB登録処理] → [終了]
```

### 変更後
```
[開始] → [請求マスター登録画面表示] → [取引先コード入力] → [取引先マスター検索] → [部品コード入力] → [部品マスター検索] → [数量入力] → [単価入力] → [単位入力] → [自動計算処理(数量×単価)] → [登録ボタン押下] → [必須項目チェック(単価/数量/単位)] → [DB登録処理] → [終了]
```

### 主な変更点
1. **単位入力ステップの追加**
   - 数量入力と単価入力の後に「単位入力」ステップを追加
   - 単位は必須入力項目として設定

2. **自動計算処理の追加**
   - 数量または単価が入力/変更された時点で、金額を自動計算するステップを追加
   - 計算式: 金額 = 数量 × 単価
   - 金額の手入力ステップを削除（自動計算に置き換え）

3. **入力チェック処理の強化**
   - 必須項目（単価、数量、単位）のチェック処理を追加
   - 未入力項目がある場合のエラーハンドリングフローを追加

## 2. 請求書出力フロー

### 変更前
```
[開始] → [請求書出力画面表示] → [出力条件設定] → [出力ボタン押下] → [請求マスターデータ取得] → [請求書フォーマット生成] → [印刷/PDF出力] → [終了]
```

### 変更後
```
[開始] → [請求書出力画面表示] → [出力条件設定] → [出力ボタン押下] → [請求マスターデータ取得] → [単位情報取得] → [請求書フォーマット生成(単位情報含む)] → [印刷/PDF出力] → [終了]
```

### 主な変更点
1. **単位情報の取得処理の追加**
   - 請求マスターから単位情報を取得するステップを追加
   - 請求書フォーマット生成時に単位情報を含める処理を追加

2. **請求書フォーマットの変更**
   - 請求書フォーマットに単位列を追加
   - 数量列の後に単位列を配置

## 3. エラー処理フロー

### 必須項目未入力時のエラーフロー
```
[入力チェック] → [必須項目(単価/数量/単位)の未入力を検出] → [該当項目の背景色変更] → [エラーメッセージ表示] → [入力画面に戻る]
```

### 数値項目の不正値入力時のエラーフロー
```
[入力チェック] → [数値項目(単価/数量)の不正値を検出] → [該当項目の背景色変更] → [エラーメッセージ表示] → [入力画面に戻る]
```

## 4. 自動計算処理フロー

### 数量または単価変更時の自動計算フロー
```
[数量または単価の入力/変更] → [入力値の数値チェック] → [数量×単価の計算実行] → [金額項目に計算結果を表示]
```

## 5. システム全体への影響

本改修による請求処理フローの変更は、以下のシステム機能に影響します：

1. 請求マスター登録機能
   - 画面レイアウト変更（単位項目追加、金額項目のグレーアウト）
   - 入力制御の追加（自動計算、必須入力チェック）

2. 請求書出力機能
   - 出力帳票レイアウト変更（単位列追加）
   - データ取得処理の変更（単位情報の取得追加）

3. データベース
   - 請求マスターテーブルのスキーマ変更（単位カラム追加）
