# 受払システム_画面定義書(17.請求マスター登録) 改修版

## 概要
本ドキュメントは、「受払システム_画面定義書(17.請求マスター登録).xlsx」の改修版です。単位項目の追加、数量×単価の自動計算機能、金額項目のグレーアウト設定、必須入力項目の背景色設定に関する変更を反映しています。

## 1. 画面レイアウト

### 変更前
```
+--------------------------------------------------+
| 請求マスター登録                                 |
+--------------------------------------------------+
| 取引先コード: [     ] [検索]  取引先名: [表示]  |
| 部品コード:   [     ] [検索]  部品名:   [表示]  |
| 数量:         [     ]                            |
| 単価:         [     ]                            |
| 金額:         [     ]                            |
|                                                  |
|           [登録]  [戻る]                         |
+--------------------------------------------------+
```

### 変更後
```
+--------------------------------------------------+
| 請求マスター登録                                 |
+--------------------------------------------------+
| 取引先コード: [     ] [検索]  取引先名: [表示]  |
| 部品コード:   [     ] [検索]  部品名:   [表示]  |
| 数量:         [     ] *                          |
| 単位:         [     ] *                          |
| 単価:         [     ] *                          |
| 金額:         [     ] (自動計算・編集不可)       |
|                                                  |
|           [登録]  [戻る]                         |
+--------------------------------------------------+
* 必須入力項目
```

### 主な変更点
1. **「単位」項目の追加**
   - 数量項目の後に「単位」項目を新規追加
   - テキストボックスとして実装
   - 最大文字数: 10文字
   - 入力必須項目として設定（「*」マークで表示）

2. **金額項目の表示変更**
   - 金額項目を編集不可（グレーアウト）に変更
   - 「(自動計算・編集不可)」の注釈を追加

3. **必須入力項目の明示**
   - 数量、単位、単価を必須入力項目として「*」マークで表示

## 2. 項目定義

| 項目名 | 種類 | 桁数 | 入力形式 | 必須 | 初期値 | 備考 |
|--------|------|------|----------|------|--------|------|
| 取引先コード | テキストボックス | 10 | 半角英数 | ○ | なし | 取引先マスターと連携 |
| 部品コード | テキストボックス | 15 | 半角英数 | ○ | なし | 部品マスターと連携 |
| 数量 | テキストボックス | 10 | 数値 | ○ | なし | 小数点以下3桁まで |
| **単位** | **テキストボックス** | **10** | **文字列** | **○** | **なし** | **例: 個、箱、kg など** |
| 単価 | テキストボックス | 10 | 数値 | ○ | なし | 小数点以下2桁まで |
| 金額 | テキストボックス | 12 | 数値 | - | なし | 自動計算・編集不可 |

※ 太字は今回の改修で追加・変更された項目

## 3. 機能定義

### 3.1 自動計算機能

| 機能名 | 説明 | トリガー | 処理内容 |
|--------|------|----------|----------|
| **金額自動計算** | **数量と単価から金額を自動計算** | **数量または単価の入力/変更時** | **金額 = 数量 × 単価** |

### 3.2 入力チェック機能

| チェック項目 | 条件 | エラー時の動作 |
|--------------|------|----------------|
| 必須項目チェック | 取引先コード、部品コード、数量、**単位**、単価が未入力 | 該当項目の背景色を変更（薄い赤色 #FFDDDD）し、エラーメッセージを表示 |
| 数値項目チェック | 数量、単価に数値以外が入力された場合 | 該当項目の背景色を変更し、エラーメッセージを表示 |
| 桁数チェック | 各項目の入力桁数が超過した場合 | 該当項目の背景色を変更し、エラーメッセージを表示 |

### 3.3 入力制御

| 項目名 | 制御内容 |
|--------|----------|
| 金額 | 編集不可（グレーアウト）に設定 |
| 数量、単位、単価 | 未入力時は背景色を変更（薄い黄色 #FFFFDD）して入力を促す |

## 4. 画面遷移

### 4.1 基本遷移

| ボタン | 遷移先 | 条件 |
|--------|--------|------|
| 登録 | 登録完了メッセージ表示後、同画面 | 入力チェックに問題がない場合 |
| 戻る | メニュー画面 | - |

### 4.2 エラー時遷移

| エラー内容 | 遷移先 | 備考 |
|------------|--------|------|
| 入力チェックエラー | 同画面 | エラーメッセージを表示し、該当項目の背景色を変更 |
| DB登録エラー | 同画面 | エラーメッセージを表示 |

## 5. メッセージ定義

| メッセージID | メッセージ内容 | 表示条件 |
|--------------|----------------|----------|
| MSG001 | 登録が完了しました。 | 登録処理成功時 |
| MSG002 | 必須項目が未入力です。 | 必須項目未入力時 |
| MSG003 | 数値項目に不正な値が入力されています。 | 数値項目に数値以外が入力された場合 |
| **MSG004** | **単位は必須入力項目です。** | **単位が未入力の場合** |

## 6. DB連携

| 処理内容 | 対象テーブル | 操作 |
|----------|--------------|------|
| 請求マスター登録 | M_SEIKYU（請求マスター） | INSERT/UPDATE |
| 取引先情報取得 | M_TORI（取引先マスター） | SELECT |
| 部品情報取得 | M_BUHIN（部品マスター） | SELECT |

## 7. 改修影響範囲

本改修による影響範囲は以下の通りです：

1. 画面レイアウト変更
   - 単位項目の追加
   - 金額項目のグレーアウト設定

2. 機能追加
   - 数量×単価の自動計算機能
   - 必須入力項目（単価、数量、単位）の背景色設定

3. DB連携変更
   - 請求マスターテーブルへの単位情報の登録
