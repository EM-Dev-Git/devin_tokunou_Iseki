# IbUkeharai アプリケーション調査報告書
**調査日**: 2025年6月2日  
**調査ブランチ**: 20250602_investigative  
**調査対象**: Devin_Iseki リポジトリ

## 概要
IbUkeharai（受払システム）は、井関物流向けの請求書書き換えおよび支払処理を行う日本のビジネスアプリケーションです。Visual Basic .NETで開発されており、Windows Forms技術を使用したデスクトップアプリケーションです。

## アプリケーション構成

### プロジェクト構造
```
IbUkeharai.sln (Visual Studio ソリューション)
├── IbUkeharai (メインアプリケーション)
├── DitCore (コアフレームワークライブラリ)
└── DitDataAccess (データアクセス層)
```

### 技術仕様
- **開発言語**: Visual Basic .NET
- **フレームワーク**: .NET Framework 4.6
- **UI技術**: Windows Forms
- **レポート**: Microsoft ReportViewer
- **データベース**: SQL Server (DitDataAccess経由)
- **外部連携**: Excel COM、FTPサーバー、CSV処理

## 技術的特徴

### カスタムコントロール (DitCore)
1. **CclBuhinCode** - 部品コード専用入力コントロール
   - マスク形式: "aaaa-aaa-aaa-aa"
   - IME無効化
   - 自動フォーマット機能

2. **CclDateTimePicker** - 拡張日付選択コントロール
   - NULL値対応
   - 日本語フォーマット対応 ("yyyy年M月")
   - カスタム描画機能

### 設定クラス
1. **DataBaseInfo** - データベース接続情報
   - サーバー名、DB名、ユーザーID、パスワード

2. **FtpServerInfo** - FTPサーバー設定
   - プライマリ・バックアップURL対応
   - テストモード機能

3. **FtpServer_KeppinInfo** - 欠品処理専用FTP設定
   - 松山、熊本、重信の拠点別ファイル管理

## アプリケーション起動フロー

1. **Program.vb** - エントリーポイント
   - ミューテックスによる重複起動防止
   - 例外ハンドリング設定
   - TopForm起動

2. **TopForm.vb** - メインフォーム
   - ロゴ画面表示 (frmLogo)
   - 設定値検証
   - 拠点別表示 (松山/熊本/部品松山)
   - 子フォーム管理

3. **BaseMenuForm.vb** - メニューフォーム基底クラス
   - 共通UI要素
   - 子フォーム表示制御
